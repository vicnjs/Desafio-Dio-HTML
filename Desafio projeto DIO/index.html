Historia dos computadores

A palavra “computador” vem do verbo “computar” que, por sua vez, significa “ calcular”. Sendo assim, podemos pensar que a criação de computadores começa na idade antiga, já que a relação de contar já intrigava os homens.

Por volta de 1640, o matemático francês Pascal inventou a primeira máquina de calcular automática. Essa máquina foi sendo aperfeiçoada nas décadas seguintes até chegar no conceito que conhecemos hoje.

A primeira máquina mecânica programável foi introduzida pelo matemático francês Joseph-Marie jacquard. Tratava-se de um tipo de tear capaz de controlar a confecção dos tecidos através de cartões perfurados.

George Boole (1815-1864) foi um dos fundadores da lógica matemática. Essa nova área da matemática, se tornou uma poderosa ferramenta no projeto e estudo de circuitos eletrônicos e arquitetura de computadores. Já no século XIX, o matemático inglês Charles Babbage criou uma máquina analítica que, a grosso modo, é comparada com o computador atual com memória e programas.Através dessa invenção, alguns estudiosos o consideram o “Pai da Informática”.

Assim, as máquinas de computar foram cada vez mais incluindo a variedade de cálculos matemáticos ( adção, subtração, divisão, multiplicação, raiz quadrada, logaritmos, etc).

O que são clientes

Cliente na linguagem computacional e quem consome os serviços de uma entidade servidora geralmente pela rede global de internet por meio de um:celular, desktop, notbook, tablet e relogios.

Em uma estrutura Cliente-Servidor o cliente usufrui do produto final distribuido pelo servidor.

O que são servers

Servers ou servidores são computadores locais ou em grandes Data Centers, basicamente servidores servem para armazenar dados e distribuir recursos para o cliente final.

Em um servidor existe tanto o Hardware quanto o Software e podemos listar os diversos tipos de servidores como:

  • Aplicação.
  • Arquivos.
  • Banco de dados.
  • Midia.
  • Email.
  • Backup.
  • Proxy
  • Web.

HTML basico

A definição de HTML e HyperText Markeup Languagee o componente base da web que permite a contrulção e inserção de textos, imagens, videos etc, por meio de hypertexto.

Outras tecnologias são usadas para descrever a aparencia/apresentação como CSS e JAVASCRIPT

O HTML usa "Marcação" para anotar texto, imagem e outros conteúdos para exibição em um navegador da Web.

Tags

A marcação HTML inclui "elementos" especiais conhencidos como TAGS,o nome de um elemento dentro de uma tag é insensível a maiúsculas e minúsculas. Isto é, pode ser escrito em maiúsculas, minúsculas ou um mistura. Por exemplo, a tag title pode ser escrita como title, TITLE ou de qualquer outra forma.

As tags atribuem funcionalidades aos elemntos de texto como deixa-los em Negrito, Sublinhado, Italico, Marcados, etc.Podendo tambem ser usado a junção de elementos como:Aqui foi usado a junção dos quatro elementos anteriores.

Lista ordenada

Entre as diversas Tags presentes no HTML existe as de listagem como: Lista ordenada e Lista não ordenada

Lista ordenada e usada para listagem de coisas onde a ordem faz diferança enumerando os elementos, exemplo:

  1. A
  2. B
  3. C
  4. D

A lista não ordenada e semelhante a anterior porem sem a enumerção, geralmente usada para casos onde a ordem dos elemntos não faz diferença, exemplo:

  • A
  • B
  • C
  • D
